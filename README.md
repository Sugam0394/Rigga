# 🚀 Rigga

Rigga Trip is a WhatsApp-based accountability system that helps you stay disciplined using roast, social pressure, and habit tracking.

Set a goal, check in daily, and if you fail — get roasted or exposed to your friends/family.

---

## ⚙️ Tech Stack

- Node.js
- Express.js
- MongoDB (Mongoose)
- Twilio WhatsApp API

---

## 🧠 How It Works

1. User joins via WhatsApp
2. Sets a daily habit/goal
3. Gets daily check-in reminders
4. If completed → gets praise  
5. If missed → gets roasted 😈  
6. Repeated misses → witness/friends notified

---

## 📦 Features (MVP)

- WhatsApp bot integration
- Daily check-ins
- Roast & praise system
- Streak tracking
- Witness (accountability partner)
